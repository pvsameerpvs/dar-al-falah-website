# Dar Al Falah Website

A light-theme 5-page website starter built with:

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- MUI (Material UI)

## Pages

- Home
- About
- Products
- Services
- Contact

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Notes

- The contact form is UI-only for now.
- You can later connect it to email, WhatsApp, or a backend API.
- Replace the placeholder product descriptions with real product photos, brands, and catalogue details.
